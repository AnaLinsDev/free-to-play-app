# Free to Play App

Application developed to help users discover and explore a wide variety of free-to-play games. Games can be filtered by genre, making it easier to find titles that match the user's interests.

When selecting a game, users can access detailed information, including preview images, game descriptions, minimum and recommended system requirements, and other relevant details.

Authenticated users can also add games to their wishlist, allowing them to keep track of their favorite titles and quickly access games they are interested in playing.


-----

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm build
$ npm start

# generate static project
$ npm generate
```
